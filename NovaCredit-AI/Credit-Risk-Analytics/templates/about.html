ℹ️ About This Project

NovaCredit AI — architecture & workflow.

📊 Dataset

Trained on banking_risk_analytics_dataset.csv — 5,000 loan applicant records spanning demographic, employment, and financial attributes, with a binary loan_default target label.

🧠 Machine Learning Models

Three classifiers were evaluated in Model_Training.ipynb: Logistic Regression, Random Forest, and MLP/KNN variants. The production system serves predictions using the trained Random Forest Classifier (random_forest_model.pkl), selected for its strong balance of accuracy and interpretability via feature importance.

Logistic Regression Random Forest ✅ (production) K-Nearest Neighbors MLP Neural Network

🛠️ Technologies Used

Python Flask SQLite scikit-learn Chart.js HTML5 / CSS3 / JavaScript ReportLab (PDF)
